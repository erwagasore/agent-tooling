# Contributing to agent-tooling

Thanks for your interest in contributing!

## Getting Started

1. Fork the repo and clone your fork.
2. Create a branch from `main`: `git checkout -b feat/your-feature`
3. Install dependencies and verify the project builds locally.

## Commit Messages

See [AGENTS.md](AGENTS.md#commits) for commit conventions.

## Pull Requests

- Fill in the PR description: what changed, why, and how to test.
- Link related issues (e.g., `Closes #42`).
- See [AGENTS.md](AGENTS.md#merge-strategy) for merge rules.

## Reporting Issues

- Search existing issues before opening a new one.
- Include steps to reproduce, expected vs actual behaviour, and environment details.

## Code of Conduct

Be respectful and constructive. Harassment or abusive behaviour will not be tolerated.
